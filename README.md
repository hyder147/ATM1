
# ATM1 - ATM Management System

A console-based and backend-focused ATM (Automated Teller Machine) simulation built with **C# and .NET**. It demonstrates core banking operations with proper user authentication, account management, and transaction handling.

## ✨ Features
- User Login with Card Number and PIN
- Check Account Balance
- Cash Withdrawal
- Cash Deposit
- Fund Transfer between accounts
- Transaction History
- Admin Panel (for managing accounts)
- Input validation and error handling

## 🛠 Tech Stack
- **Language**: C# 
- **Framework**: .NET (Console Application)
- **Database**: SQL Server + Entity Framework Core (if used) / In-memory storage
- **Architecture**: Layered Architecture (Services, Models, Repository)

## 🚀 How to Run Locally

1. Clone the repository
   ```bash
   git clone https://github.com/hyder147/ATM1.git
   ```

2. Navigate to the project folder
   ```bash
   cd ATM1
   ```

3. Open the solution in Visual Studio or run via terminal
   ```bash
   dotnet restore
   dotnet run
   ```

4. Follow the console instructions (Enter Card Number and PIN to login)
